## Instructions de génération

1. **Style général:** Toutes les images doivent avoir un style cohérent - professionnel, élégant, avec une esthétique haut de gamme
2. **Aspect ratio:** 4:3 ou 16:10 (pour s'adapter à la galerie)
3. **Résolution:** Minimum 1200x900px (idéalement 1600x1200px ou plus)
4. **Cohérence:** Les intérieurs doivent être cohérents mais variés (différentes maisons, différents styles)
5. **Invités:** Varier les personnes d'une image à l'autre pour montrer la diversité
6. **Lumière:** Privilégier la lumière naturelle avec des ambiances chaleureuses
7. **Qualité:** Style magazine lifestyle/food photography professionnel

## Notes techniques

- Les images seront utilisées dans une galerie défilante
- Dimensions affichées: 400x300px (avec object-fit: cover)
- Format recommandé: JPG haute qualité
- Compression: Optimiser pour le web tout en gardant une qualité visuelle élevée
