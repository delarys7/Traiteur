# Instructions pour les images de la galerie Chef à domicile

## Structure des fichiers

Les 10 images doivent être placées dans le dossier `/public/images/chef-domicile/` avec les noms suivants :

### Ligne supérieure (défilement gauche → droite)
1. `kitchen-prep-1.jpg` - Préparation en cuisine haut de gamme
2. `dinner-party-1.jpg` - Table dressée avec invités
3. `chef-presenting-1.jpg` - Chef présentant un plat
4. `wine-service-1.jpg` - Service du vin
5. `dessert-prep-1.jpg` - Dessert élaboré

### Ligne inférieure (défilement droite → gauche)
6. `kitchen-prep-2.jpg` - Cuisine moderne avec préparation
7. `reception-1.jpg` - Réception avec invités différents
8. `chef-serving-1.jpg` - Chef au service
9. `wine-tasting-1.jpg` - Dégustation de vin
10. `main-course-1.jpg` - Finalisation d'un plat principal

## Génération des images

Consultez le fichier `IMAGE_PROMPTS.md` pour les prompts détaillés à utiliser avec votre générateur d'images IA préféré (DALL-E, Midjourney, Stable Diffusion, etc.).

## Spécifications techniques

- **Format:** JPG
- **Résolution minimale:** 1200x900px
- **Résolution recommandée:** 1600x1200px ou plus
- **Aspect ratio:** 4:3 ou 16:10
- **Qualité:** Optimisée pour le web (compression modérée)
- **Style:** Cohérent entre toutes les images - professionnel, élégant, haut de gamme

## Placement

Une fois les images générées, placez-les dans :
```
/public/images/chef-domicile/
```

Le code est déjà configuré pour utiliser ces images automatiquement.
